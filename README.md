# PLM-DBPPred

What is PLM-DBPPred?
=============
A DNA binding protein prediction tool based on proteinBERT and attention.

Env
=============
Install ProteinBERT environment https://github.com/nadavbra/protein_bert.
* tensorflow (2.4.0)
* tensorflow_addons (0.12.1)
* numpy (1.20.1)
* pandas (1.2.3)
* h5py (3.2.1)
* lxml (4.3.2)
* pyfaidx (0.5.8)
